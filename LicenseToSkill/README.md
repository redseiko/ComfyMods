# LicenseToSkill

*Lose less skill on death and increase skill loss prevention timer.*

## Instructions

### Features

  * Increases 'no skill loss' status effect from 10 minutes to 20 minutes.
  * Reduces skill loss on death from 5% across all skills to 1%.
